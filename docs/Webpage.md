# Webpage

The DS has a web server that can be accessed at `http://localhost:6768`. This web server can only be accessed from localhost.

# Purposes

## Logs
The web server can be used to access driver station logs. These can be accessed by either clicking on the "Access Logs" page on the root webpage, or going to `/logfiles`. That will then let you download any log files.

## Robot Dashboar Status
The web server can be used to access the dashboard interface over websockets/http. See [Dashboard Interface](DashboardInterface.md) for more information.
